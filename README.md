# UTS_PEMROWEB

Repository ini berisi tugas **UTS Pemrograman Berbasis Web** berupa halaman web sederhana yang menampilkan biodata.

## Isi Project
- **index.html** — halaman utama berisi biodata sederhana.
- **style.css** — file untuk mempercantik tampilan halaman.
- **script.js** — menampilkan pesan sambutan ketika halaman dibuka.

## Cara Menjalankan
1. Download atau clone repository ini.
2. Buka file `index.html` di browser.
3. Akan muncul pesan sambutan dan tampilan biodata sederhana dengan nama *Faldy*.
